# Var, Let, and Const

- addition of ES6: let, const, can be used for variable declaration
- var: scope of var: globally scoped, or function/locally scope
- var variables can be re-declared and updated
- Hoisting of var: Javascript mechanism where variables and function declarations are moved to the top of their scope before code execution
- let is block scoped, can be updated, but not re-declared
- Const: maintain constant varlues. share some similarities with let declarations
- const cannot be updated or re-declared
